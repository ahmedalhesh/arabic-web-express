Create a full Arabic web application using TypeScript + Express + SQLite + React + TailwindCSS.

🏷️ اسم التطبيق:
نظام إدارة تراخيص البرامج

🧠 وصف المشروع:
أريد إنشاء تطبيق ويب بسيط لإدارة تراخيص البرامج التي أبيعها أو أوزعها.
عند شراء المستخدم لبرنامج، يتم إنشاء رقم تسلسلي (Serial Number) له داخل هذا التطبيق.
برامجي المكتوبة بلغة Python ستتصل بالتطبيق عبر REST API للتحقق من السيريال وتأكيد التفعيل.

💾 قاعدة البيانات:
استخدم SQLite (ملف database.db)
أنشئ جدولًا باسم licenses يحتوي على الحقول التالية:

| الحقل | الاسم بالعربية | النوع | الملاحظات |
|--------|----------------|--------|-------------|
| serial_number | رقم السيريال | TEXT PRIMARY KEY | السيريال الخاص بالمستخدم |
| active | نشط | BOOLEAN | هل التفعيل نشط؟ |
| device_id | رقم الجهاز | TEXT | معرف الجهاز الذي تم التفعيل عليه |
| activation_date | تاريخ التفعيل | TEXT | تاريخ التفعيل |
| status | الحالة | TEXT | صالح / منتهي / موقوف / غير مفعّل |
| notes | الملاحظات | TEXT | أي ملاحظات إضافية |

🔐 تسجيل الدخول (للمسؤول فقط):
- اسم المستخدم: `aalhesh`
- كلمة المرور: `ah123m123ed`
- بعد تسجيل الدخول، يمكنني إدارة التراخيص (إضافة / تعديل / حذف / بحث)

🧩 واجهة المستخدم (React + TailwindCSS):
- اللغة: العربية بالكامل (RTL)
- صفحة تسجيل الدخول تحتوي على:
  - حقل اسم المستخدم
  - حقل كلمة المرور
  - زر "دخول"
- بعد الدخول تظهر لوحة التحكم وتحتوي على:
  - عنوان أعلى الصفحة: "نظام إدارة تراخيص البرامج"
  - جدول التراخيص بالأعمدة التالية:
    ▪ رقم السيريال  
    ▪ نشط  
    ▪ رقم الجهاز  
    ▪ تاريخ التفعيل  
    ▪ الحالة  
    ▪ الملاحظات  
    ▪ أزرار (تعديل / حذف)
  - زر “إضافة ترخيص جديد”
  - حقل بحث عن السيريال أو الحالة
  - زر لتصدير البيانات إلى CSV أو Excel
  - زر “تسجيل خروج”

⚙️ REST API (للبرامج المكتوبة بالـ Python):
أنشئ نقطة تحقق رئيسية:
**GET /api/check?serial=XXXXX&device=YYYYY**

يجب أن يعيد الرد بصيغة JSON مثل:

{
  "found": true,
  "valid": true,
  "status": "صالح",
  "active": true,
  "serial_number": "XXXXX",
  "device_id": "YYYYY"
}

وإذا لم يتم العثور على السيريال أو كان موقوفًا:
{
  "found": false,
  "valid": false,
  "status": "غير موجود أو غير صالح"
}

🧱 المتطلبات الفنية:
- خادم Express مكتوب بـ TypeScript.
- واجهة React بـ TypeScript مع TailwindCSS.
- قاعدة بيانات SQLite (database.db)
- حماية الوصول بلوحة تحكم باستخدام Session أو JWT.
- الواجهة RTL بالعربية بالكامل.
- تصميم بسيط وأنيق.
- جاهز للنشر على Cloudflare Pages أو Cloudflare Workers.

اعرض التاريخ بصيغة "يوم/شهر/سنة" (مثل: 22/10/2025)

✅ الهدف النهائي:
إنشاء تطبيق ويب باللغة العربية لإدارة تراخيص البرامج، مع لوحة تحكم للإدارة، وواجهة REST API يمكن لبرامجي المكتوبة بالـ Python الاتصال بها للتحقق من السيريال وتفعيله.
